1. посмотреть, где я - pwd                                                                                                               
 2. создать папку - mkdir foldername                                                                                        
 3. зайти в папку - cd foldername                                                                                               
 4. создать 3 папки - mkdir foldername1 foldername2 foldername3                                           
5. зайти в любую папку - cd foldername                                                                                 
6. создать 5 файлов - touch filename1.txt filename2.txt filename3.txt filename4.js filename5.js                                                                                                                                              7. создать 3 папки - mkdir foldername4 foldername5 foldername6                                         
 8. вывести список содержимого папки - ls, ls -la, ls -l, ls -a                                                
 9. открыть txt файл - vim filename.txt                                                                                   
10. написать что-нибудь в txt файл - I insert                                                                            
 11. сохранить и выйти - :wq                                                                                                                 
12. выйти из папки на уровень выше - cd ..                                                                                   
 13. переместить 2 файла в любую папку - mv filename.txt filename.js foldername       
14. скопировать 2 файла в любую папку - cp filename1.txt filename2.txt foldername                                                                                                                                               15. найти файл по имени - find -name "*.txt", find ./foldername -name "*.txt" (в определенной папке)                                                                                                                    
16. просмотреть содержимое в реальном времени - grep шаблон (то, по чему ведется поиск) filename.txt                                                                                                           
 17. вывести несколько первых строк из текстового файла - head -n filename.txt                      
18. вывести несколько последних строк из текстового файла - tail -n filename.txt               
19. просмотреть содержимое длинного файла - less filename.txt                                         
 20. вывести дату и время - date                                                                                               
21. отправить http запрос на сервер - curl
